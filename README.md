

# Overview

The User Management Web Application is a user-friendly platform that allows administrators to manage user information effectively. It provides essential functionalities to view, update, and delete user details, all accessible through a secure login system. The app is designed with simplicity and efficiency in mind, making it ideal for managing a user database.

# Features
**1. Secure Login** 
- Users must authenticate using a username and password.
- Role-based access control ensures only authorized users can manage the system.
**2. View Users**
- A centralized dashboard displays a list of all users in the system.
**3. Update User Information**
- Edit and update the following user details:
    - First Name
    - Last Name
    - Email Address
- Real-time validation ensures data integrity (e.g., valid email format).
**4. Delete Users**
- Permanently remove users from the system with a simple interface.

# Installation 

- Prerequisites
    - ReactJs
    - Redux

# steps

**1. Clone the repository**

- git clone <repository-url>  
- cd user-management

**2.Install dependencies**

- npm install

**3.Start the application**

- npm run dev

# Usage
- **Login:** Access the application by entering your credentials.
- **View Users:** Navigate to the user dashboard to see a list of all registered users.
- **Update Details:** Click on a user record to edit their information.
- **Delete Users:** Use the delete button next to a user record to remove them.
